Is this software easy enough for me to install?

It is very easy. It requires you a basic ability to follow my instructions (read the install manual carefully by letter and follow step-by-step).

NOTE: Installing phpBB and especialy modifications supposes that one who does it is experienced with database, PHP, HTML, web hosting and administration.

Does this add-on only work with MySQL?

This add-on uses phpBB's DBAL so it can access whatever your database.

But we developped this add-on on MySQL and PostgreSQL backend only so we don't try it on other DBs.

I want to show post`s karma at viewtopic, how i can do it?

Just use {postrow.POST_KARMA} at viewtopic template. For example: <dd><strong>{L_POST_KARMA}:</strong> {postrow.KARMA}</dd>

How to upgrade from previous versions?

For upgrading Karma MOD from old versions you must read /contrib/x.y.z.to.a.b.c.xml where 'x.y.z' is version from which you upgrading and 'a.b.c' is next version, for example: 1.0.0b3.to.1.0.0.xml

Also links for upgrade guides placed at main install guide.

So you need to upgrade manually step-by-step from version to version
